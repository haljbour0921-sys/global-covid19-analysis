# global-covid19-analysis
This project analyzes worldwide COVID-19 trends using publicly available data. It includes data cleaning, exploratory data analysis, and visualizations to identify case trends, death rates, and the most affected countries. Tools used include Excel/Tableau and data analysis techniques to generate actionable insights.
Global COVID-19 Data Analysis
📌 Project Overview
This project analyzes global COVID-19 data to identify trends in confirmed cases, deaths, and regional impact. The goal is to demonstrate data cleaning, exploratory data analysis (EDA), and data visualization skills suitable for entry-level Data Analyst roles.
The analysis focuses on answering key questions about how the pandemic evolved over time and which countries and regions were most affected.
🎯 Objectives
Clean and prepare raw COVID-19 data
Analyze global case trends
Identify top affected countries
Calculate death rates
Create visual dashboards
Extract meaningful insights

📊 Analysis Performed
1. Total Cases by Country
Identified countries with the highest confirmed case counts.
2. Global Trend Over Time
Created a line chart showing how cases increased and changed throughout the pandemic.
3. Death Rate Analysis
Calculated and analyzed death rate trends across countries and over time.
4. Regional Comparison
Compared case distribution across different regions.

💡 Key Insights
Global case numbers increased in multiple waves.
Certain highly populated countries reported the highest totals.
Death rates changed over time due to improved treatments and vaccination efforts.
Regional differences were significant in case distribution.
